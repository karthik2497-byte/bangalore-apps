Gmail Auto-Detect Setup

To use Gmail auto-detect, you need a Google Cloud OAuth Client ID:

  1. Go to console.cloud.google.com
  2. Create a project and enable the Gmail API
  3. Create OAuth 2.0 credentials (Web application type)
  4. Add your domain as an authorized JavaScript origin
  5. Paste the Client ID in Settings → Gmail Integration
